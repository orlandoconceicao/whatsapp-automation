# WhatsApp Automation

---

## Sobre o projeto

Este projeto foi desenvolvido como parte de um processo seletivo para estágio em desenvolvimento Python.

O sistema realiza a leitura de contatos armazenados no **Supabase** e envia mensagens personalizadas automaticamente via **Z-API (WhatsApp)**.

---

## Setup da Tabela

O projeto utiliza estrutura de banco de dados para armazenar logs, contatos e mensagens automatizadas.

As principais tabelas (ou modelos) seguem a lógica:

- `id` → identificador único do contato
- `nome_contato` → nome do contato salvo no sistema
- `telefone` → número de telefone utilizado para envio de mensagens

## Estrutura SQL

```sql
CREATE TABLE contatos (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    nome_contato VARCHAR(100),
    telefone VARCHAR(20)
);
```

## Dependências

Todas as dependências do projeto estão centralizadas no arquivo requirements.txt.

Instalação:

```bash
pip install -r requirements.txt
```
---

## Principais bibliotecas:

- requests → consumo de APIs externas
- supabase → integração com banco de dados Supabase
- python-dotenv → gerenciamento de variáveis de ambiente

## Como Rodar

### 1. Clonar o repositório

```bash
git clone https://github.com/orlandoconceicao/whatsapp-automation
cd whatsapp-automation
```

### 2. Criar ambiente virtual (opcional)

```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS
```
---

### 3. Configurar variáveis de ambiente

cp .env.example .env

---

### 4. Executar o projeto

python main.py
